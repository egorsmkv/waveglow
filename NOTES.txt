
My GPU: NVIDIA RTX A4000
Python: 3.8.13

Install system packages:

    sudo apt update
    sudo apt install python3-pip

   
Install PyTorch:

    pip install torch==1.7.1+cu110 torchaudio==0.7.2 -f https://download.pytorch.org/whl/torch_stable.html

Clone the repo

  git clone https://github.com/egorsmkv/waveglow
  cd waveglow
  git submodule init
  git submodule update

Install deps

    pip3 install -r requirements.txt

Fix errors

     pip install numba==0.48 resampy==0.3.1

Download data

    wget https://huggingface.co/datasets/Yehor/ukrainian-tts-lada/resolve/main/dataset_lada_trimmed_22khz.zip
    unzip dataset_lada_trimmed_22khz.zip
    
    mv dataset_lada_trimmed/accept data
    
 Generate train lists
 
    ls data/*.wav | tail -n+10 > train_files.txt
    ls data/*.wav | head -n10 > test_files.txt

Train

    mkdir checkpoints
    python train.py -c config.json
